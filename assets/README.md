# Assets Directory

This directory contains all the game/project assets organized by type.

## Structure

- **art/**: Visual assets including sprites, textures, UI elements, and concept art
- **text/**: Text-based assets including scripts, descriptions, lore, and metadata
- **audio/**: Audio assets including music, sound effects, voice recordings, and ambient sounds

## Organization Guidelines

1. Keep assets organized in their respective subdirectories
2. Use descriptive filenames
3. Include metadata files where applicable
4. Follow naming conventions consistently
5. Keep source files when possible (e.g., .psd, .xcf for art assets)

## File Naming Convention

Use lowercase with hyphens for separators:
- Good: `player-sprite-idle.png`
- Good: `background-music-level1.ogg`
- Good: `character-dialogue-intro.txt`

Avoid spaces and special characters in filenames.
