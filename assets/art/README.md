# Art Assets

Visual assets for the project.

## Subdirectories

- **sprites/**: Character sprites, objects, and animated elements
- **textures/**: Background textures, surface materials, and patterns
- **ui/**: User interface elements, buttons, icons, and menus
- **concepts/**: Concept art, sketches, and reference materials

## Supported Formats

- **Preferred**: PNG (for sprites/UI), JPEG (for textures/concepts)
- **Source files**: PSD, XCF, SVG (keep originals when possible)
- **Vector graphics**: SVG, AI

## Guidelines

1. Use transparent backgrounds for sprites and UI elements (PNG)
2. Keep resolution consistent within categories
3. Include source files for future editing
4. Optimize file sizes for final assets
5. Use descriptive filenames indicating purpose/state
