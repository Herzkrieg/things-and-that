# Audio Assets

Audio content for the project.

## Subdirectories

- **music/**: Background music, themes, and musical compositions
- **sfx/**: Sound effects, impact sounds, and short audio clips
- **voice/**: Voice acting, narration, and spoken dialogue
- **ambient/**: Environmental sounds, atmosphere, and ambient loops

## Supported Formats

- **Preferred**: OGG Vorbis, WAV (uncompressed), MP3
- **High quality**: FLAC, WAV (for source/master files)
- **Streaming**: OGG, MP3 (for large music files)

## Guidelines

1. Use consistent sample rates (44.1kHz or 48kHz recommended)
2. Keep bit depth appropriate (16-bit for final, 24-bit for source)
3. Normalize audio levels appropriately
4. Use compression formats for distribution (OGG/MP3)
5. Keep uncompressed masters when possible
6. Include loop points for ambient/music tracks
7. Consider file size vs. quality trade-offs
