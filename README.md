# Things and That - Asset Repository

A organized collection of art assets, text assets, and audio assets for creative projects.

## Repository Structure

```
assets/
├── art/                 # Visual assets
│   ├── sprites/         # Character sprites, objects, animations
│   ├── textures/        # Background textures, materials
│   ├── ui/             # User interface elements
│   └── concepts/       # Concept art, sketches
├── text/               # Text-based content
│   ├── scripts/        # Dialogue, cutscenes
│   ├── descriptions/   # Item descriptions, character bios
│   ├── lore/          # Background story, world-building
│   └── metadata/      # Configuration files, data structures
└── audio/             # Audio content
    ├── music/         # Background music, themes
    ├── sfx/           # Sound effects
    ├── voice/         # Voice acting, narration
    └── ambient/       # Environmental sounds
```

## Getting Started

1. Browse the `assets/` directory for organized content
2. Check individual README files in each subdirectory for specific guidelines
3. Follow the naming conventions described in `assets/README.md`
4. Keep source files when possible for future editing

## Contributing

- Use descriptive filenames with hyphens (e.g., `player-sprite-idle.png`)
- Organize assets in appropriate subdirectories
- Include source files when applicable
- Follow format guidelines in each category's README
